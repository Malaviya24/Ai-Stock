# 🔷 STEP 2 — CALCULATE REQUIRED RATIOS

---

## 1️⃣ Price / Book Value

If Yahoo provides P/B directly, use it.

Else:

```javascript
pb = price / bookValue
```

---

## 2️⃣ Price / EPS (PE)

```javascript
pe = price / eps
```

---

## 3️⃣ Net Sales Per Share

```javascript
netSalesPerShare = annualSales / sharesOutstanding
```

Then:

```javascript
p_nsps = price / netSalesPerShare
```

---

## 4️⃣ Price / RONW

```javascript
p_ronw = price / ronw
```

Note:
RONW should be in %.
If RONW = 15% → use 15, not 0.15.

---

# 🔷 STEP 3 — LTVI CALCULATION

Final formula:

```javascript
ltvi = pb + pe + p_nsps + p_ronw
```

Lower LTVI = Better value stock.
