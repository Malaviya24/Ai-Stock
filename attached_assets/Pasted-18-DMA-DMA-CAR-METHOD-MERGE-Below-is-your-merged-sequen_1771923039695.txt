# 📊 PHASE 2 – SMART CAR LOGIC

For each stock passed from Phase 1:

Fetch:

* Minimum 365 daily candles

---

## STEP 1 – Detect 52-Week High

Find highest close in last 365 days.

Store:

```
highDate
high52Week
```

---

## STEP 2 – Calculate Cumulative Average (CAR)

From highDate to latest date:

For each day:

```
CAR_day_n = Average of closing prices from highDate to current day
```

Store CAR values in array.

---

## STEP 3 – Detect 10-Day Consecutive Rising CAR

Check last 10 CAR values:

Condition:

```
CAR[n] > CAR[n-1]
CAR[n-1] > CAR[n-2]
...
(10 consecutive days rising)
```

If TRUE:

```
CAR_Signal = "STRONG_BUY"
```

Else:

```
CAR_Signal = "AVOID"
```

Even one break → AVOID.
