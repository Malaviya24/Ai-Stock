# 📐 STEP 4 – CALCULATE RSI (Backend Logic)

```javascript
export const calculateRSI = (data, period = 14) => {
  let gains = 0;
  let losses = 0;

  for (let i = 1; i <= period; i++) {
    const change = data[i].close - data[i - 1].close;
    if (change > 0) gains += change;
    else losses -= change;
  }

  let avgGain = gains / period;
  let avgLoss = losses / period;

  const rs = avgGain / avgLoss;
  return 100 - (100 / (1 + rs));
};
```

---

# 📈 STEP 5 – CALCULATE 124 DMA

```javascript
export const calculateDMA = (data, days = 124) => {
  const slice = data.slice(-days);
  const sum = slice.reduce((acc, item) => acc + item.close, 0);
  return sum / days;
};
```

---

# 📊 STEP 6 – SRTV CALCULATION

```javascript
export const calculateSRTV = (price, dma124) => {
  return price / dma124;
};
```

### SRTV Rules:

* `< 0.96` → Undervalued (Start SIP)
* `> 1.12` → Overvalued (Book Profit)

# 🚀 STEP 7 – DARVAS BREAKOUT LOGIC

```javascript
export const calculateBreakout = (data) => {
  const last5Days = data.slice(-5);
  const highestHigh = Math.max(...last5Days.map(d => d.high));
  return highestHigh;
};
```

Buy when:

```
Current Price > Highest High (5 Days)
```

Target:

```
Entry Price × 1.06
```

# 🤖 STEP 9 – AI SCORING LAYER (MVP VERSION)

Instead of full ML initially, use weighted scoring model.

## Scoring Model:

| Factor            | Weight |
| ----------------- | ------ |
| RSI Oversold      | 30%    |
| SRTV Undervalued  | 30%    |
| Breakout Strength | 20%    |
| Volume Surge      | 20%    |

Example:

```javascript
export const calculateScore = (rsi, srtv, breakout) => {
  let score = 0;

  if (rsi < 35) score += 30;
  if (srtv < 0.96) score += 30;
  if (breakout) score += 20;

  return score;
};
```
