Purpose:

IF:

```
52-week LOW date is older
```

Meaning:
Stock moved from LOW → HIGH

```
ZONE = GREEN
```

Allowed:
• Start new Darvas trades
• Start SIP

## 📌 BIOS RED ZONE LOGIC

IF:

```
52-week HIGH date is older

52-week LOW date is recent
```

Meaning:
Stock moved from HIGH → LOW

Then:

```
ZONE = RED
```

Rule:
❌ Do NOT start new trades
