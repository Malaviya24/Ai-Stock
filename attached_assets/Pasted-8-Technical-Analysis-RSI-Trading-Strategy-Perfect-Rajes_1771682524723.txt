# 🔷 STEP 1 — RSI Calculation

Use standard 14-period RSI:

```javascript
RSI = 100 - (100 / (1 + RS))
```

Where:

```javascript
RS = AvgGain(14) / AvgLoss(14)
```

Use Wilder smoothing.

---

### Fetch Data

```javascript
import yahooFinance from "yahoo-finance2";

const data = await yahooFinance.historical("HDFCBANK.NS", {
  period1: "2024-01-01",
  period2: "2025-01-01",
  interval: "1d"
});
```

---

### Calculate RSI

```javascript
import { RSI } from "technicalindicators";

const closes = data.map(d => d.close);

const rsiValues = RSI.calculate({
  values: closes,
  period: 14
});
```
